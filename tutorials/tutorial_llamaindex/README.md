# LlamaIndex — Learn in 60 Minutes

This tutorial is part of the "Learn X in 60 Minutes" series.

It introduces the core concepts and API of **LlamaIndex**, and demonstrates how to build a complete Retrieval-Augmented Generation (RAG) system using real-world data.

## Contents

- `llamaindex.API.ipynb`  
  Core API components and architectural overview.

- `llamaindex.example.ipynb`  
  End-to-end RAG example using Project Gutenberg texts.

- `llamaindex.API.md`  
  Summary of the API components and abstractions.

- `llamaindex.example.md`  
  Overview of the applied example workflow.

## Models Used

- Embeddings: `sentence-transformers/all-MiniLM-L6-v2`
- LLM: `TinyLlama/TinyLlama-1.1B-Chat-v1.0`

Both models run locally and require no API keys.

## Dataset

Public domain books from Project Gutenberg:

- Pride and Prejudice
- The Adventures of Sherlock Holmes
