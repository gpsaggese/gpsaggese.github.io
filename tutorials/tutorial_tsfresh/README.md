# tsfresh Tutorial

Learn tsfresh — automated time series feature extraction — in 60 minutes.

## Quick Start

- `cd tutorials/tutorial_tsfresh`
- `./docker_build.sh`
- `./docker_jupyter.sh`
- Open the notebooks in order:
  1. `tsfresh.API.ipynb` — core API: long format, feature extraction,
     feature selection, pipeline integration, custom calculators.
  2. `tsfresh.example.ipynb` — end-to-end human activity recognition with
     the UCI HAR dataset using tsfresh + RandomForest.

For Docker build system details see the
[project template README](../../class_project/project_template/README.md).
