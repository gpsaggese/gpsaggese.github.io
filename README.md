# Drug-Drug Interaction Prediction (Project 3)

## 🎯 Objective
Predict potential interactions between drugs based on their molecular structures represented as graphs using Graph Neural Networks (GNNs).

## 🧠 Overview
- Transform molecular structures into graph representations (atoms and bonds)
- Implement a GNN model (Graph Attention Network - GAT)
- Train to predict drug-pair interactions and evaluate using ROC-AUC
- Analyze molecular features that drive interactions

## 📊 Dataset
We are using the **Drug-Drug Interaction Dataset** from [Kaggle](https://www.kaggle.com/).  
A small sample (`sample.csv`) is included for exploration.
