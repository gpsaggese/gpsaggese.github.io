# Employee Attrition — Example (End-to-End)

This example shows:
1. Load IBM HR Attrition CSV.
2. Fit model via the API wrapper.
3. Evaluate metrics (Accuracy, F1, ROC AUC).
4. Explain predictions with LIME for a few instances.
5. Save artifacts and a short results table.
Dataset suggestion: Kaggle “IBM HR Analytics Employee Attrition & Performance”.
