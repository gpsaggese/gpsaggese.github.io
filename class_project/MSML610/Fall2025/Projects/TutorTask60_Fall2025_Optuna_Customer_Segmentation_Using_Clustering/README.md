# Optuna Customer Segmentation Using Clustering

## Setup & Run Instructions

### Using Docker:
1. Build the container:
./docker_build.sh

2. Start Jupyter:
./docker_jupyter.sh

3. Open browser to: http://127.0.0.1:8888

### Project Files:
- `Optuna_utils.py` - Data loading and cleaning pipeline
- `test_data_loading.ipynb` - Test notebook with 4,338 customers, 392,692 transactions
- `online_retail.xlsx` - UCI Online Retail dataset
