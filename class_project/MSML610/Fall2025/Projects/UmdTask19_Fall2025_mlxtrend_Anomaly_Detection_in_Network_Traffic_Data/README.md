# Anomaly Detection in Network Traffic Data
This repository contains the code and documentation for the MSML610 Fall 2025 midterm project.
- `anomaly_API.ipynb`: Preprocessing and feature selection pipeline.
- `anomaly_example.ipynb`: EDA, model training (RandomForest, XGBoost), and evaluation.
- `anomaly_utils.py`: Helper utilities for loading UNSW-NB15 dataset.
- `outputs/`: Contains generated feature selection JSON and plots.
- `data/`: Extracted CSVs from the UNSW dataset.
