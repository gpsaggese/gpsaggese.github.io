## Introduction
### Overview
Tetrad is a Java library of various interfaces for exploring causal explanations of data. With the py-tetrad python interface, we can access all of Tetrad's functionality, from creating graphs of relationships within data to simulating data from constructed models.

### Goals
By the end of this tutorial, you'll be able to:
- Understand the basic functionality and usage of Tetrad
- Set up JPype for use with py-tetrad
- Create relationship graphs using Tetrad
- Explore causal explanations of data
