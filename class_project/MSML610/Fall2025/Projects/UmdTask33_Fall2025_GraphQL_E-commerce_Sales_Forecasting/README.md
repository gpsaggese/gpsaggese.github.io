# UmdTask33 – GraphQL E-commerce Sales Forecasting

This folder contains the project files for
UmdTask33_Fall2025_GraphQL_E-commerce_Sales_Forecasting:

- GraphQL.API.md / .ipynb – API design and implementation
- GraphQL.example.md / .ipynb – end-to-end usage example
- GraphQL_utils.py – shared project utilities
- utils_data_io.py – data loading and saving helpers
- utils_post_processing.py – metrics, plots, post-processing
- Dockerfile – containerization for the project
