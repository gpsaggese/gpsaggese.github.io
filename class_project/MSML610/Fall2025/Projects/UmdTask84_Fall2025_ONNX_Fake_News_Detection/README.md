### 🎯 Project Overview
This project aims to detect fake news using a neural network model exported to the **ONNX (Open Neural Network Exchange)** format for efficient inference across platforms.

### 📁 Folder Structure
- `ONNX_Fake_News_Detection_utils.py` → Utility functions  
- `utils_data_io.py` → Data loading & preprocessing  
- `utils_post_processing.py` → Result analysis  
- `.API.*` → ONNX model API and explanation  
- `.example.*` → Example usage demonstration  
- `Dockerfile` & scripts → Containerized setup
