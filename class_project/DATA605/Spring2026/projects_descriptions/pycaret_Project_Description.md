# PyCaret

## Description
- PyCaret is an open-source, low-code machine learning library in Python that
  simplifies the end-to-end machine learning workflow.
- It provides a unified interface for various machine learning tasks including
  classification, regression, clustering, and anomaly detection.
- PyCaret automates many aspects of the modeling process, including data
  preprocessing, feature selection, and hyperparameter tuning, allowing users to
  focus on insights rather than code.
- The library integrates seamlessly with popular libraries such as Scikit-learn,
  Matplotlib, and Plotly, making it versatile for data visualization and
  analysis.
- PyCaret supports model comparison and ensembling, enabling users to evaluate
  multiple models quickly and select the best performing one.
- It is designed to be user-friendly, allowing users to build models with just a
  few lines of code, making it ideal for both beginners and experienced
  practitioners.

## Project Objective
The goal of the project is to predict housing prices based on various features
such as location, size, and amenities. Students will explore the dataset,
preprocess the data, build multiple machine learning models using PyCaret, and
evaluate their performance to identify the best model for price prediction.

## Dataset Suggestions
1. **Kaggle Housing Prices Dataset**
   - **Source**: Kaggle
   - **URL**:
     [Housing Prices - Advanced Regression Techniques](https://www.kaggle.com/c/house-prices-advanced-regression-techniques/data)
   - **Data Contains**: Features about houses in Ames, Iowa, including sale
     price, square footage, number of bedrooms, and neighborhood.
   - **Access Requirements**: Free to use with a Kaggle account.

2. **Open Government Data - Housing Affordability**
   - **Source**: Data.gov
   - **URL**:
     [Housing Affordability Data](https://www.data.gov/dataset/housing-affordability)
   - **Data Contains**: Data on housing prices, median income, and affordability
     metrics across various U.S. cities.
   - **Access Requirements**: No authentication required.

3. **Zillow Home Value Index (ZHVI)**
   - **Source**: Zillow
   - **URL**: [Zillow Data](https://www.zillow.com/howto/api/APIOverview.htm)
   - **Data Contains**: Historical home values, rental prices, and property
     characteristics for various regions.
   - **Access Requirements**: Free access with a provided API key.

4. **Kaggle California Housing Prices**
   - **Source**: Kaggle
   - **URL**:
     [California Housing Prices](https://www.kaggle.com/c/california-housing-prices/data)
   - **Data Contains**: Features related to housing prices in California,
     including geographic and socio-economic factors.
   - **Access Requirements**: Free to use with a Kaggle account.

## Tasks
- **Data Exploration**: Use PyCaret to load and analyze the dataset, identifying
  key features and relationships.
- **Data Preprocessing**: Automatically preprocess the data using PyCaret
  functions to handle missing values, categorical encoding, and feature scaling.
- **Model Training**: Utilize PyCaret's classification or regression module to
  train multiple models and compare their performance.
- **Model Evaluation**: Evaluate the models using metrics such as R-squared,
  Mean Absolute Error (MAE), and visualizations to understand model performance.
- **Model Tuning**: Fine-tune the best-performing model using PyCaret's
  hyperparameter tuning capabilities.
- **Final Report**: Document the findings, model performance, and insights
  gained throughout the project in a comprehensive report.

## Bonus Ideas
- Compare the performance of traditional machine learning models with ensemble
  methods or deep learning models.
- Implement feature engineering techniques to create new features and assess
  their impact on model performance.
- Explore the effects of different data imputation techniques on model accuracy.
- Create a dashboard using Plotly or Streamlit to visualize the predictions and
  insights interactively.
