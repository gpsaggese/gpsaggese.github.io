# LlamaIndex

## Description
- LlamaIndex is a powerful framework designed for building applications that
  leverage large language models (LLMs) by providing a structured way to connect
  data sources with LLMs.
- It enables users to create custom data indices that can be queried using
  natural language, making it easier to extract and manipulate information from
  diverse datasets.
- The tool supports various data sources, including structured data (like CSVs
  and databases) and unstructured data (like documents and text), allowing for
  versatile applications.
- LlamaIndex includes features for data ingestion, indexing, and querying, which
  simplifies the process of integrating LLMs into data science projects.
- It offers a user-friendly API that facilitates the development of applications
  without needing extensive knowledge of LLMs or natural language processing.

## Project Objective
The goal of the project is to build a question-answering system that can extract
relevant information from a dataset of historical events. The system will
optimize for accuracy in answering user queries based on the indexed data.

## Dataset Suggestions
1. **Dataset Name**: Historical Events Dataset
   - **Source**: Kaggle
   - **URL**:
     [Historical Events Dataset](https://www.kaggle.com/datasets/benroshan/historical-events)
   - **Data Contains**: A collection of significant historical events with
     dates, descriptions, and categories.
   - **Access Requirements**: Free to download upon creating a Kaggle account.

2. **Dataset Name**: Wikipedia Articles on Historical Events
   - **Source**: Wikipedia API
   - **URL**:
     [Wikipedia API Documentation](https://www.mediawiki.org/wiki/API:Main_page)
   - **Data Contains**: Articles and summaries of various historical events,
     which can be fetched programmatically.
   - **Access Requirements**: Public API with no authentication required.

3. **Dataset Name**: World History Timeline
   - **Source**: GitHub Repository
   - **URL**:
     [World History Timeline](https://github.com/yourusername/world-history-timeline)
   - **Data Contains**: A timeline of world history with events listed
     chronologically, including dates and descriptions.
   - **Access Requirements**: Open access, can be cloned or downloaded from
     GitHub.

4. **Dataset Name**: The World Factbook
   - **Source**: CIA World Factbook
   - **URL**: [CIA World Factbook](https://www.cia.gov/the-world-factbook/)
   - **Data Contains**: Information on countries, including historical context
     and significant events.
   - **Access Requirements**: Publicly accessible and free to use.

## Tasks
- **Data Ingestion**: Load the chosen dataset into LlamaIndex, ensuring that it
  is properly formatted for indexing.
- **Index Creation**: Create an index of the historical events data using
  LlamaIndex to facilitate efficient querying.
- **Query Development**: Develop a set of natural language queries that users
  can use to retrieve information about historical events.
- **Model Integration**: Integrate a pre-trained language model to interpret
  user queries and fetch relevant data from the index.
- **Evaluation**: Assess the accuracy of the responses generated by the system
  and refine the indexing and querying process based on feedback.

## Bonus Ideas
- Implement a feature that allows users to ask follow-up questions based on the
  initial answers provided.
- Compare the performance of different pre-trained language models (e.g., GPT-3
  vs. BERT) in terms of accuracy and response time.
- Create a visualization of the historical events data to provide context for
  the answers generated by the system.
