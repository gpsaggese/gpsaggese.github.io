# Streamlit

## Description
- Streamlit is an open-source app framework specifically designed for creating
  and sharing data applications quickly and easily.
- It allows users to build interactive web applications for data science
  projects with minimal coding, using only Python.
- Streamlit supports real-time updates and allows for seamless integration of
  various data visualization libraries like Matplotlib, Plotly, and Altair.
- The tool is ideal for prototyping machine learning models and sharing results
  with stakeholders in a user-friendly format.
- With features like session state management and customizable layouts,
  Streamlit enables developers to create dynamic applications tailored to user
  inputs.

## Project Objective
The goal of this project is to develop an interactive web application that
predicts house prices based on various features such as location, size, and
number of bedrooms. Students will optimize the model's accuracy and create a
user-friendly interface for inputting house details and displaying predicted
prices.

## Dataset Suggestions
1. **Kaggle - House Prices: Advanced Regression Techniques**
   - URL:
     [House Prices Dataset](https://www.kaggle.com/c/house-prices-advanced-regression-techniques/data)
   - Contains: Historical house prices and various features (e.g., square
     footage, number of bedrooms, location).
   - Access Requirements: Free registration on Kaggle.

2. **UCI Machine Learning Repository - Boston Housing Dataset**
   - URL: [Boston Housing Data](https://archive.ics.uci.edu/ml/datasets/Housing)
   - Contains: Information about housing in Boston, including crime rates,
     property tax rates, and housing values.
   - Access Requirements: Open access without registration.

3. **Kaggle - California Housing Prices**
   - URL:
     [California Housing Prices Dataset](https://www.kaggle.com/c/california-housing-prices/data)
   - Contains: Data on housing prices in California, including features like
     median income, housing age, and location.
   - Access Requirements: Free registration on Kaggle.

4. **Open Government Data - NYC Housing Data**
   - URL:
     [NYC Housing Data](https://data.cityofnewyork.us/Housing-Development/Housing-Development-Data-2019/8v9d-8w8g)
   - Contains: Data on housing developments in NYC, including project types,
     locations, and completion dates.
   - Access Requirements: Open access without registration.

## Tasks
- **Data Exploration**: Load the chosen dataset and perform exploratory data
  analysis (EDA) to understand key features and relationships.
- **Data Preprocessing**: Clean the dataset by handling missing values, encoding
  categorical variables, and scaling numerical features as needed.
- **Model Selection**: Choose an appropriate machine learning model (e.g.,
  Linear Regression, Random Forest) for predicting house prices.
- **Model Training and Evaluation**: Train the selected model on the training
  dataset and evaluate its performance using metrics such as RMSE or R².
- **Streamlit App Development**: Build a Streamlit application that allows users
  to input house features and displays the predicted price in real-time.
- **Deployment**: Optionally deploy the Streamlit app on platforms like
  Streamlit Sharing or Heroku for wider access.

## Bonus Ideas
- Implement additional features such as price distribution visualization or
  feature importance analysis to enhance the application.
- Compare the performance of multiple models (e.g., Linear Regression vs.
  Gradient Boosting) and display the results in the app.
- Challenge students to integrate a map visualization showing house prices by
  location using libraries like Folium or Plotly.
