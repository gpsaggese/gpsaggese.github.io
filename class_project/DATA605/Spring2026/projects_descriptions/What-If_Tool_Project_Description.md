# What-If Tool

## Description
- The What-If Tool is an interactive visual interface designed for analyzing
  machine learning models without the need for coding.
- It allows users to explore model performance, visualize data distributions,
  and conduct what-if analyses by modifying input features and observing changes
  in predictions.
- Users can import datasets, visualize them in various formats, and compare
  different models side by side.
- The tool supports various machine learning tasks, including classification and
  regression, and integrates seamlessly with TensorFlow and other ML frameworks.
- It provides capabilities for feature importance analysis and debugging model
  predictions, making it ideal for educational purposes and model evaluation.

## Project Objective
The goal of this project is to build a machine learning classification model
that predicts whether a customer will churn based on their usage patterns and
demographic information. Students will optimize for accuracy and
interpretability of the model using the What-If Tool.

## Dataset Suggestions
1. **Telco Customer Churn Dataset**
   - **Source**: Kaggle
   - **URL**:
     [Telco Customer Churn](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)
   - **Data Contains**: Customer demographics, account information, and services
     used, along with a binary churn label.
   - **Access Requirements**: Free to use; sign up for a Kaggle account to
     download.

2. **Customer Segmentation Dataset**
   - **Source**: Kaggle
   - **URL**:
     [Customer Segmentation](https://www.kaggle.com/datasets/khushalsharma/customer-segmentation)
   - **Data Contains**: Customer profiles, purchase history, and segmentation
     labels.
   - **Access Requirements**: Free to use; sign up for a Kaggle account to
     download.

3. **Bank Marketing Dataset**
   - **Source**: UCI Machine Learning Repository
   - **URL**:
     [Bank Marketing](https://archive.ics.uci.edu/ml/datasets/bank+marketing)
   - **Data Contains**: Information about bank clients, including demographic
     data and marketing campaign results.
   - **Access Requirements**: Publicly available without authentication.

## Tasks
- **Data Import and Exploration**: Load the selected dataset into the What-If
  Tool and perform initial exploratory data analysis (EDA) to understand the
  data distribution and feature relationships.
- **Model Training**: Use the What-If Tool to train a classification model
  (e.g., logistic regression, decision tree) on the dataset and evaluate its
  performance using accuracy and confusion matrix.
- **What-If Analysis**: Utilize the What-If Tool to modify input features for
  specific customers and observe how these changes affect churn predictions,
  helping to understand model behavior.
- **Feature Importance Assessment**: Analyze and visualize the importance of
  different features on the model's predictions and discuss the implications for
  business strategies.
- **Model Comparison**: If time permits, compare different models (e.g.,
  decision tree vs. random forest) using the What-If Tool's side-by-side
  comparison feature.

## Bonus Ideas
- Implement hyperparameter tuning for the classification models and analyze how
  it affects performance.
- Explore the impact of synthetic data generation on model predictions and
  robustness.
- Conduct a detailed error analysis on misclassified instances to identify
  patterns or biases in the model.
