# Dolibarr

## Description
- Dolibarr is an open-source ERP and CRM software designed for small and
  medium-sized enterprises (SMEs).
- It provides a suite of integrated applications for managing various business
  processes, including customer relationship management, invoicing, project
  management, and inventory control.
- Key features include user-friendly dashboards, customizable modules, and
  support for multiple languages and currencies.
- Dolibarr supports a modular architecture, allowing users to enable or disable
  features according to their specific business needs.
- It offers RESTful API endpoints, making it easier to integrate with other
  applications and perform data analysis.

## Project Objective
The goal of the project is to analyze sales data from a fictional company using
Dolibarr's API to predict future sales trends. Students will develop a machine
learning model to forecast monthly sales, optimizing for accuracy and
interpretability.

## Tasks
- **API Exploration**: Familiarize yourself with the Dolibarr API, focusing on
  endpoints related to sales data retrieval.
- **Data Collection**: Use the API to extract sales data and combine it with the
  chosen dataset for analysis.
- **Data Preprocessing**: Clean and preprocess the data, including handling
  missing values and formatting timestamps.
- **Feature Engineering**: Create relevant features for the sales prediction
  model, such as seasonal indicators and product categories.
- **Model Development**: Select and implement a machine learning model (e.g.,
  Linear Regression, Random Forest) to predict future sales.
- **Model Evaluation**: Assess model performance using metrics like Mean
  Absolute Error (MAE) and R-squared, and visualize results.

## Bonus Ideas
- Implement additional machine learning algorithms and compare their performance

- Extend the project by incorporating external factors that may influence sales,
  such as marketing campaigns or economic indicators.
- Create a dashboard using visualization libraries (e.g., Matplotlib, Seaborn)
  to present findings and predictions interactively.
